INTRODUCTION

Transcription factors (TFs) play crucial roles in transcriptional regulation processes, and the relationships between TFs and their target genes are essential in the control of cell-specific gene expression and the occurrence of diseases.

How to Use the TFTG?

Browse

The “Browse” page is organized as an interactive table that allows users to quickly search for TFs and customize filters according to “TF Family”, “TF Class” and “TF Name”. Users can click the “CSV” to download the basic information of TFs. To view details of a TF, users click on "TF".

Search

TFTG provides multi-type query modes to facilitate the users to query TF-target genes flexibly.The users can determine the scope of TF-target genes through "Search by TF", "Search by target gene" and "Search by genomic region" .


In "Search by TF"，the users can search TF-target genes by inputting the TF name. After submitting,the users can obtain the detailed information of on the input TF, including TF overview, TF target gene network, table of TF-target genes, detailed regulatory information ofTF-targets,TF expression, disease information and annotation of TF.

In the"Search by target gene"mode, the users can select one of 14 methods, set a weight threshold and input a gene name of interest to search for TFs that regulate this gene. The weights are calculated by counting the number of methods that determine the regulatory relationship ofTFs to target genes. Then,TFTG returns the detailed information of these TFs in the result table.

ln the"Search by genomic region"mode,with the input of a genomic region and selection of a method, TFTG returns the details of all TF-target genes associated with the input region in the result table.

Analysis

TFTG designs four types of analyses for the TF-target network to elucidate the regulatory mechanisms and function of TF including TF gene set enrichment analysis, TF downstream regulatory analysis, Gene upstream regulatory analysis and Pathway enrichment analysis.


TF gene set enrichment analysis.

With the input of the gene set of interest, TFTG returns the list of TFs that are significantly enriched based on target genes using the hypergeometric test. TFTG also provides various parameters to facilitate the users to filter the results, including TF-target genes identification method, the P-value/FDR, number of gene and sample of interest.

TF downstream regulatory analysis.

By inputting TFs of interest and selecting specificity or non-specificity network, TFTG provides the regulatory network visualizations of these TFs and targets. The node properties in this network are also displayed in a table for insight into interactions.

Gene upstream regulatory analysis.

By inputting genes of interest, the users can visualize the regulatory network formed by these genes and associated TFs.

Pathway enrichment analysis.

The users can select at least one pathway source and input a gene set of interest, TFTG returns the significantly enriched pathways. The significance P-values were calculated using the hypergeometric test. In the result table, the terminal TFs of the pathway and their target genes are displayed.

Download

TFTG provides a convenient and flexible download function for TF-related files, including “TF information”, “TF ChIP-seq data information”, “TF perturbation RNA-seq data information”, “TF ChIP-seq or perturbation RNA-seq sample” and “TF downstream target gene of different methods” . Meanwhile, the database provides batch download for all TF-target genes identified by different methods. We support ".bed", ".txt" or ".zip" formats for data download.

The detailed instructions of the 4 categories and 14 sub-categories methods for identifying TF-target genes in TFTG

ChIP-seq_class: We first collected a large number of TF binding sites (TFBSs) from TF ChIP-seq datasets involving human multiple tissue/cell types from five public sources, including ENCODE, Cistrome, Remap, ChIP-Atlas and GTRD. All ChIP-seq datasets were further deduplicated by manual screening according to the unique GEO/SRA ID to avoid duplication (Supplementary Materials). For the uniformity of genome version, peaks from ChIP-seq were converted to the hg38 genome using the liftOver tool of UCSC (http://genome.ucsc.edu/cgi-bin/hgLiftOver). Finally, we identify TF-targets based on these TFBSs using 7 strategies (sub-categories prefixed with C_), including C_SE, C_TE, C_ATAC, C_Silencer, C_Promoter, C_BETA, and C_Genemapper, as detailed below:


C_SE: The letter ‘SE’ in ‘C_SE’ stands for the super-enhancers. The SEs were obtained from SEdb database developed by our group. Briefly, we curated H3K27ac ChIP-seq raw data from Roadmap, ENCODE, NCBI GEO/SRA and Genomics of Gene Regulation Project (GGR) and run the streamlined pipeline Bowtie-MACS14-ROSE to identify SE regions. In addition, we also downloaded SEs from other projects including SEA, ENdb and dbSuper. For each SE region, we used ROSE geneMapper.py to annotate downstream target genes. Then, we identified TF-target genes when the gene-associated SEs were occupied by TFBSs in matched tissues/cell types using BEDTools.


C_TE: The letter ‘TE’ in ‘C_TE’ stands for the typical enhancers. We comprehensively curated the resources provided in the existing enhancer databases, including EnhancerAtlas, FAMTOM5, HANCER, EnhancerDB, DENdb, SEdb and ENdb. For each TE region, we used ROSE geneMapper.py to annotate downstream target genes. Then, we identified TF-target genes when the gene-associated enhancers were occupied by TFBSs in matched tissues/cell types using BEDTools.


C_ATAC: The letter ‘ATAC’ in ‘C_ATAC’ stands for the accessible chromatin regions. More than 2,200 publicly available human ATAC-seq samples were manually curated from NCBI GEO/SRA by ATACdb developed by us. Then, we gained accessible chromatin regions by filtering ATAC-seq samples and running Bowtie2-MACS2. For each accessible chromatin region, we used ROSE geneMapper.py to annotate downstream target genes. Finally, we identified TF-target genes when the gene-associated accessible chromatin regions were occupied by TFBSs in matched tissues/cell types using BEDTools.


C_Silencer: The silencers were collected from SilencerDB. We used ROSE geneMapper.py to annotate downstream target genes for each silencer region. Then, we identified TF-target genes when the gene-associated silencers were occupied by TFBSs in matched tissues/cell types using BEDTools.


C_Promoter: We defined promoter regions as 2kb upstream and 2kb downstream of the transcription start sites (TSSs) of genes, which were collected from GENCODE. A gene was considered as TF target when the peaks of TF overlapped with the gene promoter region. The overlapping information was calculated using BEDTools.


C_BETA: For each TF ChIP-seq peak, we measured the potential power for the TF-target regulations using the BETA method. The beta-model score was calculated based on the number of peaks within a certain range and the distance between the peak and TSS.


C_Genemapper: We used a Python script geneMapper.py from ROSE to annotate downstream target genes for TFs. This script provided the closest, proximal and overlapping genes for TFs according to the genomic distance between TFBSs and genes.


Perturbation_class. All TF perturbation RNA-seq data were collected from the KnockTF database developed by our group. We curated these datasets from NCBI GEO and ENCODE using a list of keywords, such as ‘knockout’, ‘knockdown’, ‘siRNA’, ‘shRNA’ and ‘CRISPR’. We further traversed the title, summary and protocol of preliminary screening results to ensure data quality. Then, we mapped Ensembl IDs to gene symbols with regard to each gene expression profile and deleted genes with zero values in all samples. Then the raw expression values of gene expression profiles were processed by Log2 transformation. We further computed fold change (FC) for each gene. For datasets with more than three samples, The R package limma was used to compute the statistical significance of differential expression. We extracted differentially expressed genes under the threshold of FC ≥3/2 or FC ≤2/3 as TF-target genes.


Curate_class. The method ‘Curate’ represents literature-supported TF-target genes which are generally considered the gold standard dataset. We collected literature-supported TF-target pairs from TRRUST and CHEA3. TRRUST first manually collected Medline abstracts. Then, it used a method named sentence-based text_mining to extract text sentences that might be related to transcriptional regulation, which were finally subjected to manual curation to gain TF-target genes. The regulation in CHEA3 was derived from TF ChIP-seq and ChIP-chip experiments mined from the biomedical research literature.


Motif_class. We first obtained > 3,000 DNA binding motifs from the TRANSFAC and MEME suite deriving from the following five sources: JASPAR CORE 2020 vertebrates, Jolma2013, Homeodomain, UniPROBE and Wei2010. Next, we identify TF-targets based on these motifs using 5 strategies (sub-categories prefixed with M_), including M_SE, M_TE, M_ATAC, M_Silencer and M_Promoter, as detailed below:


M_SE: We took the TF targets as those genes associated with SEs with TF motif occurrence using FIMO software of the MEME suite.


M_TE: We took the TF targets as those genes associated with typical enhancers with TF motif occurrence using FIMO software of the MEME suite.


M_ATAC: We took the TF targets as those genes associated with accessible chromatin regions with TF motif occurrence using FIMO software of the MEME suite.


M_Silencer: We took the TF targets as those genes associated with silencer regions with TF motif occurrence using FIMO software of the MEME suite.


M_Promoter: We took the TF targets as those genes associated with promoter regions with TF motif occurrence using FIMO software of the MEME suite.
