TF gene set enrichment analysis

Method Select All





Function introduction

If users input an gene set (gene symbol), TFTG will execute hypergeometric test between the submitted gene set and target gene set of each TF.

1) Method: Select at least one method of TF target genes.

2) Genes: Input a gene symbol set.

3) Threshold: Set P-Value and FDR thresholds.

4) GeneNumber: Limit the number range of genes in TF target genes.

5) Biosample name: Select one sample biosample name or more.
